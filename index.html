The ABC Model

Welcome. This is a tool for teaching the ABC model for disputing unhelpful thoughts to improve how we feel, think and act.

The structured procedure described here is designed to be easy to follow while providing clarification on common stumbling points in practice.

This tool is not a replacement for professional help and is intended for educational purposes. Please speak to a clinician/therapist for further clarification on how the ABC model is applicable for your situation.
